# Lab 2
Implementation of the following functionalities from the cdev API: open, release, read, write, ioctl. The driver provides some basic calculator operations. Ioctls are used to:
- reset calculations
- select operation (+, -, *, /)
Example flow:
```
ioctl operation_add
write <-- 2
write <-- 10
read --> 12
```

* ```my_driver.c``` - driver implementing character device API.
* ```my_driver.h``` - separate header file with defines for ioctls, so that they can be copied to include/uapi.
* ```app.c``` - example userspace program to test the driver functionality. Should be cross-compiled for RISC-V 32-bit architecture - for example using toolchain generated by Buildroot.
